# Bank Management System

A simple banking application built using Python, Object-Oriented Programming (OOP), JSON file handling, and Streamlit.

## Features

- Create a new bank account
- Deposit money
- Withdraw money
- View account details
- Update account information
- Delete account
- Data persistence using JSON

## Tech Stack

- Python
- Streamlit
- JSON
- OOP

## Project Structure

```text
├── app.py          # Streamlit UI
├── hello.py        # Bank class and business logic
├── data.json       # Database file
├── requirements.txt
└── README.md
```

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Bank-Management-System-Python.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run app.py
```
